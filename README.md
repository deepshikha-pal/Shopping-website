# Amazon Clone - Responsive Shopping Website

## Overview

This project is a responsive Amazon Clone website built using HTML and CSS. It replicates the basic layout, design, and user interface of Amazon, providing a clean and responsive shopping experience across different devices.

## Features

* Responsive Design for Mobile, Tablet, and Desktop
* Amazon-inspired Homepage Layout
* Navigation Bar with Search Section
* Product Categories Section
* Hero Banner Section
* Product Cards and Shopping Sections
* Footer Similar to Amazon
* Clean and User-Friendly Interface

## Technologies Used

* HTML
* CSS3
